# University Lost and Found System

A web application for managing lost and found items at a university. The system allows users to report lost items, report found items, browse through listings, and submit claims for items.

## Features

- Lost item submission with details about the item, location, date, and contact information
- Found item submission with similar details and item images
- Browse and search functionality with filters for item type and date
- Claim request system for lost items that have been found
- Item status tracking (lost, found, claimed)
- Responsive design that works on all devices

## Technology Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **Data Storage**: In-memory data structures (Python dictionaries/lists)

## Getting Started

### Prerequisites

- Python 3.6 or higher

### Installation

1. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   python app.py
   ```

3. Access the application in your browser at:
   ```
   http://127.0.0.1:5000
   ```

## Project Structure

- `app.py`: Main Flask application
- `templates/`: HTML templates for pages
- `static/`: Static files (CSS, JavaScript, etc.)
  - `css/`: Style sheets
  - `js/`: JavaScript files
